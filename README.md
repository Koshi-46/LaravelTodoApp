# アプリケーション名
TodoList

## 説明
タスクの追加や更新ができるアプリ
ジャンルの選択もでき、
またそれらのタスクを更新したり削除したりできる

## URL
http://laraveltodoapp.link/todo

## 機能一覧
タスクの追加・更新・削除
